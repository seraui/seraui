# Text highlighter
The text highlighter component is a powerful tool that allows you to highlight specific text within a given text string. It is designed to be used in various scenarios, such as displaying search results, code snippets, or any other text-based content.

## How to Use

### Basic Usage

```tsx

export default function Example() {
  return (
    <p>
      Welcome to <TextHighlighter>Sera UI</TextHighlighter>

```

### Highlight Directions

Control the direction of the highlight animation:

```tsx
// Left to right (default)
<TextHighlighter direction="ltr">Left to right</TextHighlighter>

// Right to left
<TextHighlighter direction="rtl">Right to left</TextHighlighter>

// Top to bottom
<TextHighlighter direction="ttb">Top to bottom</TextHighlighter>

// Bottom to top
<TextHighlighter direction="btt">Bottom to top</TextHighlighter>
```

### Animation Transitions

Customize the animation timing and easing:

```tsx
<TextHighlighter
  transition={{
    duration: 1.2,
    delay: 0.3,
    type: 'spring',
    bounce: 0.2
  }}
>
  Custom animation
</TextHighlighter>
```

### InView Options

Configure when the highlight triggers based on viewport intersection:

```tsx
<TextHighlighter
  triggerType="inView"
  useInViewOptions={{
    once: true,        // Only animate once
    initial: false,    // Don't animate initially
    amount: 0.5,       // 50% of element must be visible
    margin: '100px'    // Trigger 100px before entering viewport
  }}
>
  Advanced in-view highlighting
</TextHighlighter>
```

### Different HTML Elements

